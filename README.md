# 1. Hospital Analytics: Management Dashboard

An interactive **Power BI dashboard** designed to monitor hospital operations—focusing on patient data, revenue tracking, disease trends, and doctor performance.

---

## 2. Short Description / Purpose

The Hospital Management Dashboard provides a centralized view of hospital data to track patient statistics, revenue flow, appointment status, and disease distribution. It helps healthcare administrators make informed decisions and improve operational efficiency.

---

## 3. Tech Stack

The dashboard was built using the following tools and technologies:<br>

➤ **Power BI Desktop** – Used for creating interactive dashboards and visual reports.<br>
➤ **Power Query** – Used for data cleaning and transformation.<br>
➤ **DAX (Data Analysis Expressions)** – Used for KPI calculations and measures.<br>
➤ **Data Modeling** – Relationships created between patient, doctor, and billing data.<br>
➤ **Python (Pandas, Matplotlib)** -  Used for initial analysis.<br>
➤ **File Formats** – `.pbix` for dashboard, `.csv` for dataset.<br>

---

## 4. Data Source

The dataset contains hospital-related information such as:

➤ Patient details (ID, count)
➤ Revenue (Total, Paid, Pending)
➤ Disease types
➤ Doctor appointments
➤ Payment methods

The data was preprocessed and structured for visualization.

---

## 5. Features / Highlights

### ▸ Business Problem

Healthcare institutions often face challenges such as:

➤ Difficulty in tracking patient count and hospital load
➤ Lack of visibility into revenue (paid vs pending)
➤ Inefficient monitoring of doctor performance
➤ Limited insights into disease trends and appointment status

---

### ▸ Goal of the Dashboard

The dashboard aims to:

➤ Provide real-time insights into hospital operations
➤ Track revenue performance and pending payments
➤ Analyze disease distribution and trends
➤ Monitor doctor appointments and efficiency
